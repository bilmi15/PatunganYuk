# PatunganYuk!

**PatunganYuk!** adalah aplikasi Android native yang dirancang untuk mempermudah pengelolaan, perhitungan, dan pembagian pengeluaran atau tagihan kelompok secara praktis. Aplikasi ini membantu memberikan transparansi keuangan yang cepat dan akurat untuk aktivitas bersama seperti makan bareng, acara kelompok, atau tagihan bulanan.

## 🚀 Fitur Utama
- **Manajemen Tagihan:** Pencatatan data tagihan kelompok dan pelacakan status pembayaran secara teratur.
- **Statistik & Visualisasi:** Halaman laporan untuk melihat ringkasan alokasi pengeluaran dan kontribusi setiap anggota.
- **Penyimpanan Data Lokal:** Menggunakan sistem database internal agar seluruh data keuangan tetap tersimpan dengan aman di dalam perangkat pengguna.
- **Antarmuka Modern:** Dibangun menggunakan komponen UI terbaru yang responsif dan mudah digunakan.

## 🛠️ Cara Instalasi
Langkah-langkah untuk menyiapkan dan membuka proyek ini di lingkungan pengembangan lokal Anda:

1. **Prasyarat:** Pastikan Anda telah menginstal [Android Studio](https://developer.android.com/studio) versi terbaru di komputer Anda.
2. **Ekstrak Proyek:** Ekstrak file zip proyek ini ke salah satu folder direktori lokal Anda.
3. **Buka di Android Studio:** 
   - Jalankan Android Studio.
   - Pilih menu **Open** atau **Import Project**.
   - Arahkan ke folder hasil ekstrak proyek ini, lalu klik **OK**.
4. **Sinkronisasi Gradle:** Biarkan Android Studio mengunduh seluruh dependensi dan pustaka (*library*) yang diperlukan secara otomatis hingga proses *Gradle Sync* selesai dengan sukses.

## 📱 Cara Penggunaan
Setelah proses sinkronisasi selesai, Anda bisa langsung menjalankan aplikasi ini dengan langkah berikut:

1. **Hubungkan Perangkat:** Sambungkan perangkat Android fisik Anda menggunakan kabel data (pastikan *USB Debugging* aktif) atau gunakan *Android Virtual Device (Emulator)*.
2. **Jalankan Aplikasi:** Klik tombol **Run** (ikon segitiga hijau `▶`) pada toolbar bagian atas Android Studio atau tekan kombinasi tombol `Shift + F10`.
3. **Alur Penggunaan:** 
   - Buka aplikasi yang telah terpasang di perangkat.
   - Mulai masukkan daftar pengeluaran baru pada halaman utama.
   - Tentukan pembagian dana antar anggota dan pantau ringkasannya pada halaman statistik yang tersedia.
